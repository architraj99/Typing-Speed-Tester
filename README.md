# Typing-Speed-Tester

Typing Speed Tester is a website built to test user's speed and accuracy.

Demo Video 
https://cdn.hackclub.com/019ecfb3-3501-7243-8295-197608672fe1/2026-06-16_14-38-00.mp4

NOTE - IF EVERY LETTER IS CORRECT AND IN GREEN COLOR AND YOU HAVE ALSO PUT THE FULL STOP THEN TIME WILL STOP AND THE RESULT WILL COME INSTANTL
IN OTHER HAND IF YOU HAVE DONE ANY ERRORS IN THE SENTENCE THEN YOU WILL HAVE TO WAIT 60 SEC FOR THE RESULT 

-----------ABOUT THE PROJECT--------------

It currently has these following feautres to monitor your great work LIVE:

1) Time    2)WPM     3)Accuracy     4)Mistakes
5)Correct Characters     6)Wrong Characters and a     7) Result Summary to conclude everything at last.

The test starts as soon as user starts typing, and ends just as the paragraph is completed, otherwise the timer runs out and you are provided with your result.

It has a feature where wrong characters get red colored and correct ones get green as you keep typing. Looks cool, right?

I might add a feature to copy paste any specfic para user wants to practice typing on...
